# dasarathi-maven-plugin

### Run Maven Command

####  Build & Package maven-plugin

``` 
mvn clean install
```

#### Run maven-plugin

``` 
# mvn groupId:artifactId:version:mojoName
mvn com.rout.dasarathi:dasarathi-maven-plugin:v1.0.0-SNAPSHOT:message
```
